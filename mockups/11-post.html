Why Flat,
Sharp,
& Bold

published 2026·04·02  ·  8 min read  ·  #design #web

There is a particular aesthetic that runs through Bungie's Marathon trilogy — a visual language born from the hard constraints of a 2.5D engine, yet so distinct that it defines the game's identity more than any plot point or mechanic. Flat textures. Sharp geometry. Bold color choices. Everything feels architectural, intentional, constructed.

This aesthetic isn't just a retro curiosity. It's a complete design philosophy that maps cleanly onto the web — onto typography, layout systems, color theory, and interaction design. And it's been hiding in plain sight, ignored by a web that chose smoothness over materiality.

The constraint forces intentionality. When you can't do realistic curves, you make geometry meaningful. When you can't do smooth gradients, you make every color choice deliberate. Every surface is a decision, not a render.

The Three Forces

The Marathon aesthetic is defined by three forces held in careful tension. Flatness — surfaces that read as planes, not as simulations of material. No gradients, no soft shadows, no glossy effects. Sharpness — the enforced orthogonal geometry of a 2.5D engine created a world of 90-degree angles, of architectural precision. Boldness — limited color palettes used with maximum impact. When you only have a few colors, every one of them has to carry weight.

Together, these forces produce something that feels more like architecture than like graphics. You're moving through a constructed space where every element justifies its presence or gets deleted.

From Game Engine to CSS

Translating this into web design: Flat: solid backgrounds, no box-shadow blur. Sharp: 2px solid borders as primary separators, geometric layouts. Bold: 2–3 accent colors on a neutral base, high contrast, typography that asserts itself.

But there's a fourth element: texture. Marathon's surfaces were flat colors with the grain of a texture map, the noise of a low-color-depth display. This is where dithering enters the vocabulary — and it's what makes a site feel like a place rather than a document.

The Texture Layer

Using SVG feTurbulence filters combined with CSS gradients and filter: contrast(), we create genuine dithered gradients — noise that transitions smoothly from dense to sparse along a color gradient. Not fake stripes. Actual Perlin noise modulated by brightness and contrast.

/* Proper dithered gradient */
.dither-grad {
  background:
    linear-gradient(to top, var(--n-teal), transparent),
    url("noise.svg");
  background-blend-mode: overlay;
  filter: contrast(3) brightness(2);
  opacity: 0.12;
}

The featured card on the homepage uses this technique. The h2 borders use it. The nav underlines use it. The noise is SVG — zero HTTP requests, ~500 bytes. It renders natively in the browser compositor at 60fps.

Typography as Architecture

Marathon's Bank Gothic — geometric sans-serif with square proportions — was used for the game's UI. It looks like it was built from steel. For the web, Squarish Sans CT carries that lineage. The display font here is Syne (geometric, bold). The body is JetBrains Mono at 0.95rem / 1.8.

The TRUEGAV pattern in the background has 5 layers: filled teal, filled green, outlined pink, outlined amber, outlined blue. Each drifts at a different speed and direction. The outlines use text-stroke — transparent fill, colored border. Some are 7rem, some 4rem. They create parallax depth.

Breaking Rules Intentionally

The h1 is 6rem and spills 6% past the left edge. The blockquote spans the full grid row — it doesn't overlap the margin, it replaces it for its duration. Cards on the homepage touch at corners (-4px margins on specific pairs). Every rule broken has a reason. None are accidents.

Where This Goes

This isn't a theme. This is a wall, a zine, a broadcast. The flat/sharp/bold aesthetic rejects drop shadows, border-radius, muddy midtones. In their place: dithered gradients, animated patterns, halftone dots, concrete grain, neon accents. A design you can feel in your fingertips.
