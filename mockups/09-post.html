Why Flat,
Sharp,
& Bold

published 2026·04·02  ·  8 min read  ·  #design #web

There is a particular aesthetic that runs through Bungie's Marathon trilogy — a visual language born from the hard constraints of a 2.5D engine, yet so distinct that it defines the game's identity more than any plot point or mechanic. Flat textures. Sharp geometry. Bold color choices. Everything feels architectural, intentional, constructed.

This aesthetic isn't just a retro curiosity. It's a complete design philosophy that maps cleanly onto the web — onto typography, layout systems, color theory, and interaction design. And it's been hiding in plain sight, ignored by a web that chose smoothness over materiality, comfort over conviction.

The constraint forces intentionality. When you can't do realistic curves, you make geometry meaningful. When you can't do smooth gradients, you make every color choice deliberate. Every surface is a decision, not a render.

The Three Forces

The Marathon aesthetic is defined by three forces held in careful tension. Flatness — surfaces that read as planes, not as simulations of material. No gradients, no soft shadows, no glossy effects. Sharpness — the enforced orthogonal geometry of a 2.5D engine created a world of 90-degree angles, of architectural precision. No organic curves. Boldness — limited color palettes used with maximum impact. When you only have a few colors, every one of them has to carry weight.

Together, these forces produce something that feels more like architecture than like graphics. You're not looking at pictures — you're moving through a constructed space where every element justifies its presence. This is the opposite of the "muddy realism" that dominated late-2000s game design.

From Game Engine to CSS

Translating this into web design means understanding what each force demands. Flat: solid color backgrounds, no box-shadow blur. Sharp: 2px solid borders as the primary visual separator. Bold: 2–3 accent colors on a neutral base, high contrast between surface levels.

But there's a fourth element: texture. Marathon's surfaces weren't just flat colors — they were flat colors with the faint grain of a texture map, the subtle noise of a low-color-depth display, the materiality of concrete and metal. This is where dithering enters the vocabulary, and it's what makes a site feel like a place.

The Texture Layer

Using SVG feTurbulence filters, we can add grain to any surface — concrete texture on backgrounds, static on interactive elements. The noise is never the point. It's the feeling of material. This page has a halftone dot grid, animated brand pattern drifting at different speeds, concrete grain, and coarse noise on code blocks.

/* Three-layer texture system */
body::before {
  /* layer 1: concrete grain */
  opacity: 0.35;
  background: var(--wall), var(--tex-concrete);
}
body::after {
  /* layer 2: halftone dots */
  opacity: 0.06;
  background: radial-gradient(
    circle, var(--chalk) 1.2px, transparent 1.2px
  );
  background-size: 8px 8px;
}
.brand-pattern {
  /* layer 3: drifting TRUEGAV text */
  opacity: 0.04;
  animation: drift 50s linear infinite;
}

The technique is lightweight — SVG data URIs, zero HTTP requests, about 500 bytes in the stylesheet. No JavaScript, no WebGL. The entire texture system renders natively. But the feel it adds is immeasurable.

Typography as Architecture

Marathon's type choices were architectural. Bank Gothic — a geometric sans-serif with square proportions — was used for the game's UI. It looks like it was built from steel beams. For the web, the open-source Squarish Sans CT carries that lineage. Paired with a monospace body font, you get the full spectrum: headers carved into the wall, body text like printed matter.

The display font here is Syne — geometric, bold, architectural. The body is JetBrains Mono at 0.95rem with 1.8 line-height — comfortable for long reading but never soft.

Breaking Rules Intentionally

Chaos must be composed. Every broken rule needs a reason. The cards on the homepage are rotated between -1 and +1.5 degrees — organic, not drunk. The blockquote pulls into the margin. The h1 spills 5% past the left edge. The code blocks push 4% past the right. These breakouts are punctuation — the rest of the reading experience is disciplined.

The animated TRUEGAV pattern in the background drifts at different speeds in different neon colors. It's subtle — 4% opacity — but it makes the wall feel alive. It's not a static background. It breathes.

Where This Goes

The flat/sharp/bold aesthetic rejects drop shadows, heavy border-radius, muddy midtones. In their place: a design you can feel. This isn't a Hugo theme or a blog template. It's a wall. And every element — the dithered borders, the drifting text, the halftone dots, the neon tags — is here because it refuses to apologize.
